# Concept

AI Fund Lab は、日本株の短期売買ルールを題材にした研究・検証プロジェクトです。投資助言ではなく、通常の実行先はPaperBrokerです。

最初のAIキャラクターは「新人ディーラー1号」です。教科書通りで慎重、判断理由を残し、ルール逸脱を嫌う新人ディーラーという設定です。

このリポジトリでは、候補抽出、スコアリング、落選理由、仮想売買、損益、feature analysis、experiment summary、売却後の振り返りまでを保存します。あとから「なぜ買ったのか」「なぜ買わなかったのか」「どの条件が利益/損失に寄与したのか」を追跡できることを重視します。

live自動売買は現在の目的ではありません。Tachibanaはread-only guarded、KabuStationはstubです。
